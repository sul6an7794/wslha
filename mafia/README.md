## البنية التقنية

- **الخادم:** Node + Express + Socket.io، بدون حسابات وبدون قاعدة بيانات — كل حالة الغرف في الذاكرة (`server/src/`).
- **الواجهة:** HTML/CSS/JS خام بدون أدوات بناء، عمود موبايل RTL (يتوسط الشاشة على الديسكتوب).
- **التشغيل:** `cd server && npm install && npm start` ثم `http://localhost:3500`. للمؤقتات السريعة أثناء التطوير: `node src/server.fast.js`.
- **الاختبارات:** `cd server && npm test` — 30 اختبارًا (`node --test`) تغطي التوزيع وكل ميكانيكيات الأدوار + جولة تكامل كاملة عبر Socket.io + إعادة الاتصال.

## القواعد (Playable v3)

- **اللاعبون 6–13**، القائد يلعب بدور. رابط دعوة `?room=CODE`.
- **إلزامي كل لعبة:** مافيا، طبيب، شيخ. **اختياري** (فرصة 40% لكل مقعد متبقٍ، نسخة واحدة قصوى): وريثة، زعيم، حرامي، عمدة، متحول، **مصارع**، **مهرج**. مافيا ثانية تلقائيًا من 11 لاعبًا. الباقي مواطنون ببطاقات نكهة بصرية عشوائية (قروي/أميرة/مهرج) بلا قدرات.
- **المصارع:** بلا قدرة ليلية (شاشة كود تمويهي كالمواطن). ينجو تلقائيًا من أول هجوم ليلي يستهدفه طوال اللعبة (مرة واحدة)، حتى بلا حماية الطبيب — والرسالة الصباحية محايدة فلا تكشف سبب النجاة.
- **المهرج:** بلا قدرة ليلية، من الخير بالانتماء لكنه **يفوز منفردًا**: يخسر لو نجا حيًا حتى نهاية اللعبة، ويفوز (كبطاقة إضافية بشاشة النهاية) لو أُقصي بالتصويت أو قُتل ليلًا في أي وقت خلال اللعبة، بصرف النظر عن نتيجة المدينة/العصابة. فحص الشيخ له عشوائي (خير أو شر) بدل نتيجة حقيقية ثابتة.
- **الأشرار** (فوز/فحص الشيخ): مافيا + وريثة + زعيم + المتحول بعد تحوله. الوريثة والزعيم لا يقتلون — يشاهدون شاشة الكود التمويهي مثل المواطن.
- **الليل (30ث، متزامن):** المافيا تقتل (لا تستهدف الأشرار؛ مع مافيتين يلزم تطابق الاختيار، والتعادل عند انتهاء الوقت = ضحية عشوائية + سطر بالسجل)، الطبيب يحمي (حتى نفسه)، الشيخ يفحص (دفتر متراكم)، الحرامي يسرق صوتًا لليوم التالي، والبقية يدخلون كود تحقق تمويهي 6 أرقام يظهر رقمًا رقمًا.
- **لعنة الوريثة:** إعدامها بالتصويت (فقط) يعطل الطبيب والشيخ الليلة التالية بشاشة «لعنة الوريثة».
- **المتحول:** إن استُهدف ليلًا بلا حماية لا يموت — ينضم سرًا للعصابة وسطر غامض بالسجل.
- **كشف القتيل:** بطاقة مجهولة «؟» بختم «قُتل» — الهوية تبقى سرًا حتى النهاية (الإقصاء بالتصويت يكشف الدور بالسجل).
- **النهار (45ث):** حدث الليلة + دفتر الشيخ (خاص) + سجل المدينة المتراكم.
- **التصويت (40ث):** صفوف بأسماء من صوّت على من، العدد الظاهر خام بينما **صوت العمدة يُحسب بوزن 2 خفيًا** (يُذكر «بنفوذ صوت العمدة المضاعف» بالسجل فقط لو رجّح). أزرار «عفو» و«إقصاء المتهم» (أغلبية الأحياء تحسم مبكرًا). تعادل موزون = لا إقصاء.
- **الدفاع (60ث):** عند متهم واضح — المتهم يدافع بلا أزرار، والبقية «تنفيذ الإقصاء» أو «تغيير الصوت» (أغلبية تحسم؛ انتهاء الوقت = تنفيذ).
- **الموت والمتفرج:** بطاقتك رمادية + السبب + «شاهد بقية اللعبة» (فلتر رمادي وتعطيل الأزرار).
- **النهاية:** بطاقات كل الناجين من الفريق الفائز بأسمائهم + السبب + «كنت [دورك] — فزت/خسرت» + «لعبة جديدة» (للقائد، بنفس اللاعبين).
- **الصوت:** زر ♪ — نغمات مولدة (Web Audio): نبضة ليل، صفارة قتل/إقصاء، رنة نجاة، تكة آخر 5 ثوانٍ، ودرون أجواء يتغير حسب المرحلة.
- المؤقتات آخر 10 ثوانٍ: عداد أحمر يومض + قرار تلقائي عند الانتهاء (عشوائي إن لا اختيار).
